FidduPay Sandbox API

Test environment for FidduPay cryptocurrency payment gateway

Available Endpoints

POST /payments

Create a new payment

{
  "amount": 100.50,
  "currency": "USDT",
  "network": "ethereum",
  "description": "Test payment",
  "metadata": { "orderId": "test-123" }
}
GET /payments/:id

Get payment status by ID

GET /payments

List all payments with optional filters

Query params: limit, offset, status, currency, network

POST /webhooks/test

Test webhook endpoint

GET /health

Health check endpoint

Quick Start

npm install
npm start

Server will run on http://localhost:3001
